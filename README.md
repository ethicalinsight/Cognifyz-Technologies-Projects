# 📊 Cognifyz Technologies Internship Projects

This repository showcases the key data science and development tasks completed during my internship at **Cognifyz Technologies** as a Python Developer Intern. The projects span the full analytics pipeline—from data cleaning and exploratory analysis to dashboard creation and machine learning.

## 🔍 Highlights
- Cleaned and preprocessed raw datasets for analysis
- Conducted exploratory data analysis (EDA) to uncover trends and patterns
- Built interactive dashboards using Power BI and Python for visual storytelling
- Extracted structured data via web scraping and API integration
- Developed predictive models using machine learning algorithms
- Delivered stakeholder-ready reports with clear documentation and insights

Each project reflects a hands-on application of data science principles, technical problem-solving, and communication tailored for real-world impact. The description of each project is as follows:-

# 1. 🍽️ Identifying Top Cuisines and Calculating Their Prevalence

This project analyzes restaurant data to identify the most frequently offered cuisines across various regions. By calculating prevalence metrics, it highlights dominant food categories and regional culinary preferences.

## 🔍 Objectives
- Count and rank cuisines by frequency
- Visualize cuisine distribution across cities
- Detect regional cuisine trends

## 🛠️ Tools Used
- Python (Pandas, Matplotlib)
- Jupyter Notebook

## 📈 Key Insights
- Indian, Chinese, and Fast Food emerged as top cuisines
- Regional specialties showed strong local dominance

# 2. 🏙️ Evaluating City-wise Restaurant Density and Rating Performance

This analysis compares restaurant density and average ratings across Indian cities to assess culinary saturation and customer satisfaction.

## 🔍 Objectives
- Calculate restaurant count per city
- Analyze average ratings and review volumes
- Identify high-performing cities

## 🛠️ Tools Used
- Python (Pandas, Seaborn)
- Power BI (Dashboarding)

## 📈 Key Insights
- High density doesn’t always correlate with high ratings
- Tier-2 cities showed surprising rating performance

# 3. 💸 Visualizing Price Range Distributions and Consumer Affordability

This project explores how restaurant pricing varies across cities and categories, offering insights into affordability and market segmentation.

## 🔍 Objectives
- Categorize restaurants by price range
- Visualize affordability across regions
- Correlate pricing with ratings

## 🛠️ Tools Used
- Python (Matplotlib, Seaborn)
- Power BI

## 📈 Key Insights
- Mid-range pricing dominates urban areas
- Affordable restaurants often receive higher ratings

# 4. 🚚 Assessing Online Delivery Impact on Rating Trends

This task investigates how online delivery availability influences customer ratings and review volumes.

## 🔍 Objectives
- Compare ratings of delivery vs. non-delivery restaurants
- Analyze review count trends
- Identify delivery-related performance patterns

## 🛠️ Tools Used
- Python (Pandas, Seaborn)
- Jupyter Notebook

## 📈 Key Insights

- Ratings are slightly higher for delivery services in metro areas

# 6. 🧪 Discovering High-performing Cuisine Combinations

This task identifies cuisine pairings that consistently receive high ratings, offering strategic insights for menu planning.

## 🔍 Objectives
- Analyze multi-cuisine restaurants
- Rank combinations by average rating
- Detect synergy between cuisine types

## 🛠️ Tools Used
- Python (Pandas, NumPy)
- Jupyter Notebook

## 📈 Key Insights

# 7. 🗺️ Mapping Restaurant Clusters Using Geolocation Data

This geospatial analysis visualizes restaurant clusters using latitude and longitude data to identify urban hotspots and underserved zones.

## 🔍 Objectives
- Plot restaurant locations on city maps
- Detect clustering patterns
- Highlight areas with high or low density

## 🛠️ Tools Used
- Python (Folium, Plotly)
- Geopandas (optional)

## 📈 Key Insights

- Peripheral zones show growth potential for new ventures
